# Carrousel d'Images en JavaScript

## Description

Ce projet consiste en la création d'un carrousel d'images interactif en JavaScript. Le carrousel permet aux utilisateurs de faire défiler des images de manière fluide avec des flèches de navigation et un défilement infini. Ce projet a été réalisé dans le cadre d'une mission pour Lucien, le directeur de Print it.



## Contexte de la Mission

La mission consistait à créer un carrousel d'images dynamique qui puisse améliorer l'expérience utilisateur sur le site de Print it. L'objectif était de permettre aux utilisateurs de visualiser les différentes images tout en offrant une navigation simple et intuitive.

## Fonctionnalités

- **Navigation par flèches** : Les utilisateurs peuvent naviguer entre les images à l'aide de flèches gauche et droite.
- **Défilement infini** : Les utilisateurs peuvent faire défiler les images de manière continue, sans interruption.
- **Changement d'images par événement** : Les images changent en réponse à des événements tels que des clics sur les flèches.
